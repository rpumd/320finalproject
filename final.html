Introduction

Television data offers valuable insights into viewership patterns, content success, and production trends, making it a critical resource for understanding what drives audience interest. This project leverages the TMDB TV dataset, a comprehensive repository of information on television shows, including attributes such as the number of episodes, seasons, vote counts, average ratings, genres, and more. By analyzing this dataset, we aim to uncover the factors that contribute to a show's success and popularity.

Specifically, this analysis is structured to address the following key questions:

  1. What are the main features influencing a show's popularity?
  2. Which features are most significant in determining whether a TV show is popular or not?
  3. Can we accurately classify whether a TV show is popular based on its features?

By addressing these questions, this project aims to provide actionable insights for media companies and streaming platforms. The findings can help these organizations optimize their content strategies, allocate resources effectively, and improve audience engagement by delivering content that aligns with viewer preferences.

Data Curation

Dataset Source:
The dataset was obtained from The Movie Database (TMDB), a comprehensive platform offering detailed information about TV shows, movies, and associated metadata.

Dataset Description:
The dataset includes attributes such as the number of seasons, episodes, ratings, popularity scores, and more. Its size and variety make it ideal for machine learning applications and exploratory data analysis.

group1 = df_cleaned[df_cleaned['number_of_seasons'] > 3]['vote_average']
group2 = df_cleaned[df_cleaned['number_of_seasons'] <= 3]['vote_average']

t_stat, p_value = stats.ttest_ind(group1.dropna(), group2.dropna())
print(f"Two-sample T-test results: T-statistic = {t_stat}, P-value = {p_value}")
if p_value < 0.05:
    print("The average rating is significantly different between the two groups (reject the null hypothesis).")
else:
    print("There is no significant difference in average ratings between the two groups (fail to reject the null hypothesis).")

Two-sample T-test results: T-statistic = 0.2581069845557183, P-value = 0.7963420104045225
There is no significant difference in average ratings between the two groups (fail to reject the null hypothesis).

The T-test comparing the average ratings of shows with more than 3 seasons versus those with 3 or fewer seasons reveals no significant difference between the two groups. This is shown by the p-value, which is 0.796. Since the p-value is greater than 0.05, the null hypothesis fails to be rejected. This suggests that the number of seasons in a show does not strongly influence its average viewer rating.

median_popularity = df_cleaned['popularity'].median()
df_cleaned['is_popular'] = (df_cleaned['popularity'] > median_popularity).astype(int)

features = ['number_of_seasons', 'number_of_episodes', 'vote_count', 'vote_average', 'episode_run_time']
target = 'is_popular'

X = df_cleaned[features]
y = df_cleaned[target]


scaler = StandardScaler()
X_scaled = scaler.fit_transform(X)


X_train, X_test, y_train, y_test = train_test_split(X_scaled, y, test_size=0.2, random_state=42)

rf_clf = RandomForestClassifier(random_state=42, n_estimators=100, max_depth=None)
rf_clf.fit(X_train, y_train)

y_pred_proba = rf_clf.predict_proba(X_test)[:, 1]
y_pred_clf = rf_clf.predict(X_test)
accuracy = accuracy_score(y_test, y_pred_clf)
print(f"\nAccuracy: {accuracy}")

# Classification Report
print("\nClassification Report:\n")
print(classification_report(y_test, y_pred_clf))
conf_matrix = confusion_matrix(y_test, y_pred_clf)
plt.figure(figsize=(8, 6))
sns.heatmap(conf_matrix, annot=True, fmt='d', cmap='Blues',
            xticklabels=['Not Popular', 'Popular'],
            yticklabels=['Not Popular', 'Popular'])
plt.title("Confusion Matrix")

Conclusion

This project helped us better understand what makes a TV show popular and how different factors contribute to its success. By analyzing the TMDB TV dataset, we identified which features matter most and used machine learning to classify shows as "popular" or "not popular." Through tools like feature importance charts, precision-recall curves, and confusion matrices, we turned raw data into meaningful insights that connect directly to real-world decisions.

One of the most interesting takeaways was how much the number of episodes influences a shows popularity. Longer-running shows clearly tend to attract more viewers, which could guide media companies to invest in content with the potential for multiple seasons. On the other hand, features like vote counts and ratings highlight the importance of audience engagement. When viewers actively participatewhether by leaving ratings, writing reviews, or casting votesthey help boost a shows visibility and overall appeal.

We also learned that even less obvious features, like the number of seasons or episode runtime, can play a role when combined with other factors. The machine learning model showed how these variables work together to paint a fuller picture of what drives popularity. The precision-recall curve and confusion matrix confirmed that the model was effective at distinguishing popular shows from those that didnt quite hit the mark.

For streaming platforms and media companies, these insights are game-changers. They offer a data-driven way to predict which shows will resonate with audiences and help refine decisions about what to produce and how to engage viewers. By focusing on shows with higher episode counts and encouraging audience interaction, companies can boost their chances of creating successful content.
